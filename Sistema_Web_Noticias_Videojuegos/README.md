# Sistema Web de Noticias de Videojuegos

## Título:
Sistema Web de Noticias de Videojuegos

## Objetivo:
Desarrollar una plataforma web que permita publicar y consultar noticias relacionadas con videojuegos.

## Alcance:
- Gestión de noticias
- Clasificación por categorías
- Panel de administración
- Base de datos relacional

## Tecnologías Utilizadas:
- Python (Flask)
- SQLite
- HTML5
- CSS3
- Git y GitHub

## Descripción General:
El sistema permite crear, visualizar y gestionar noticias de videojuegos mediante una aplicación web desarrollada en Flask. 
La información se almacena en una base de datos SQLite y se muestra en una interfaz sencilla desarrollada en HTML y CSS.
